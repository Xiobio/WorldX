Generate a 1024×1024 visual reference image — the **style anchor** for an
entire fantasy zone. This image will be passed back to every chunk in the zone
as a style reference, so it should distill the zone's full visual identity
into one beautiful, illustrative frame.

## Zone identity
{{zoneIdentity}}

## What to depict
A single panoramic vignette showing the zone's signature elements packed into
one composition: terrain, buildings, vegetation, color palette, lighting, and
atmospheric effects. **Treat this as an illustration / art print — beautiful
and atmospheric** — not a clinical engineering map.

## Hard constraints
1. NO characters or creatures.
2. NO text, labels, or captions of any language.
3. Style must be **internally consistent** — one painter, one palette, one
   light source. This image will be used to guide downstream chunks; if you
   drift across this single frame, the chunks will drift too.

## Encouragements (not constraints)
- A high-angle bird's-eye view (60-90°) is great if it fits the zone — don't
  feel trapped at strict 90° if a slightly tilted angle reads better as art.
- Show buildings with their roofs CROSS-SECTIONED if the zone uses that
  convention (no roof, low walls, interior visible) — the resulting image
  reads more like a "diorama" and lends itself to game use.
- Pack the frame: trees, water, paths, lanterns, distant mountains, weather
  / mist / falling petals — whatever makes the zone feel like itself.
- Saturation moderate so chunks have room to vary downstream.
